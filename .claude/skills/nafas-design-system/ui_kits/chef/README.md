# Nafas — Chef App UI Kit

Interactive click-through prototype of the chef-facing mobile app.

## Screens
1. **Dashboard** — Kitchen toggle, quick stats, today's orders, reviews
2. **Orders** — Filter tabs, order cards with status advancement
3. **Menu** — Item list with availability toggles, add/edit form

## Usage
Open `index.html` in any browser. Click through screens using the floating tab bar.

## Design Notes
- The chef tab bar is a floating pill (bottom-center), not a standard tab bar
- Dashboard kitchen toggle card changes to indigo background when open
- Order status pills use the 7-state color system
- Menu items show stock badge (red if ≤ 2 left)
