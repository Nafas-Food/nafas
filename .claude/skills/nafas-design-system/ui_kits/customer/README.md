# Nafas — Customer App UI Kit

Interactive click-through prototype of the customer-facing mobile app.

## Screens
1. **Welcome** — Hero + logo + CTA buttons + language toggle
2. **Sign In** — Phone + password form, error state
3. **Home** — Location header, search, hero banner, chef cards, category chips, dish cards
4. **Chef Profile** — Avatar hero, stats bar, menu items, reviews
5. **Cart** — Item list, qty steppers, address, order notes, totals
6. **Orders** — Active orders with status pills, past orders

## Usage
Open `index.html` in any browser. Click through screens using the nav buttons or in-screen interactions.

## Components
- `WelcomeScreen.jsx`
- `HomeScreen.jsx`
- `ChefProfileScreen.jsx`
- `CartScreen.jsx`
- `OrdersScreen.jsx`
